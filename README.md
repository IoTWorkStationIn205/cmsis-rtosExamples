# cmsis2-rtos示例

## 简介
使用STM32作为主控板，使用CMSIS2-RTOSAPI实现，线程创建、软件定时器等示例，采用单文件形式，不区分开发板，只要是使用STM32CUBX生成的工程均可使用。

## 使用

1. 获取代码

``` bash
git clone https://github.com/IoTWorkStationIn205/cmsis-rtosExamples.git
```

2. 将文件加入工程编译
